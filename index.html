Git

pwd: print working directory
cd: change directory
git status: displays state of working directory and staging area
git checkout main: swich to main branch
git pull origin main: pull remote main branch into local main branch
git checkout -b branch-name: creates a new branch
touch: creates new file
code . : open working directory in VS Code
git add -A: add updates to git repository
git commit -m "your message": add commit message
git pull origin main: pull base branch main to working branch
git push origin working branch
navigate to repository in GitHub
change to working branch
refresh to verify changes have been pushed
